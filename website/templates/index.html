Welcome to Speech Emotion Recognition

Speech Emotion Recognition is system where user can use A.I to predict emotions from speech.

Features

Speech Emotion Recognition System features include upload speech audio files, record speech audio and real-time speech emotion recognition. Features will be updated and new features will be added in the future.

Upload and Record Speech Audio

Upload speech audio file (.wav) or record speech audio using microphone. After uploading or recording the speech audio, user can preview the audio and predict the speech emotion using the A.I model.

Real-Time Speech Emotion Recognition

Predict speech emotion in real time. User can choose the interval of speech emotion prediction (1-5 seconds) before using the real-time SER. The system can only use device's microphone for now. Prediction using system's audio will be added in the future.
